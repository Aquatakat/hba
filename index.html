About Hoo Boy Aloha

Hoo Boy Aloha was created by me, and probably by some other people if I ever get off my arse and put it on github Google Code a GameFAQs board for some obscure game with a funny title.

It uses PHP, Ratchet and good old fashioned plain jane DOM JavaScript. I don't even abstract document.getElementById, even though I really really wanted to.

I am Ryan Rosser and I made this in 2015.

The name "Hoo Boy Aloha" is a very clever reference to a thing from the 90's, I swear.
